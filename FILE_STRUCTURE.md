### 🟢 DATA FILES

**sample_patient_reviews.csv**
```csv
patient_id,review
PATIENT_0001,Doctor was great!
PATIENT_0002,Terrible experience.
```
Purpose: Shows expected CSV format
Records: 10 samples
Use: Reference for your CSV format

**patient_reviews.csv**
- Auto-generated if missing
- Contains 1000 random reviews
- Mix of positive, negative, neutral
- Used for testing

**patient_reviews_preprocessed.csv**
- Generated by the preprocessing pipeline and used as the input to sentiment analysis
- Contains cleaned/normalized review text (column `preprocessed_text`) and preserved metadata

**patient_reviews_removed.csv**
- Contains rows dropped during preprocessing
- Includes the original `review` and a `remove_reason` field (e.g. `empty_after_cleaning`, `many_misspellings`)

**patient_reviews_with_sentiment.csv**
- Output file with analysis results
- Contains original review + analysis
- Columns: patient_id, review, polarity_score, sentiment
- Ready for further analysis

## 📦 DEPENDENCIES

**Automatically Installed By Script:**
- pandas (data handling)
- textblob (sentiment analysis)
- matplotlib (visualization)
 - nltk (tokenization, stopwords, lemmatization, POS tagging)
 - beautifulsoup4 (HTML stripping)
 - contractions (expand contractions)
 - emoji (demojize emojis)
 - pyspellchecker (basic misspelling detection)

**Installation Command:**
```bash
pip install -r requirements.txt
python -m textblob.download_corpora
```
